🏙️ Smart City App

The Smart City App is a web-based platform designed to improve communication between citizens and municipal authorities. It enables users to register complaints related to public infrastructure such as roads, water, electricity, sanitation, and more. The system helps authorities track and resolve issues efficiently, promoting transparent governance and citizen engagement.

🚀 Features

👤 User Authentication (Signup & Login)

📍 Submit Complaints with Category & Location

📸 Upload Images as Proof

📄 View Complaint Status & History

🛠️ Admin Dashboard to Manage Complaints

📨 Notifications / Status Updates

🌐 Responsive UI with Smooth User Experience
